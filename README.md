# Cmd based key bindings

I use capslock key as cmd (can be configured in keyboard system settings).

  - `cmd + ijkl`: arrow keys.
  - `cmd + h`: cmd + left.
  - `cmd + ;`: cmd + right.
  - `cmd + u`: alt + shift + left.
  - `cmd + o`: alt + shift + right.
  - `cmd + n`: delete
